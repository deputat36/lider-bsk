`https://deputat36.github.io/lidercalculator/app-v4.html`

3. Затем перенести КП, создание заказа из КП, заказы, производство и монтаж.
4. После этого обновить `crm/v4/index.html`.
5. Только после проверки переключать основной адрес CRM v4.
