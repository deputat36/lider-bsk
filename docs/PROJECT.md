# Проект РА «Лидер»

## Назначение

Проект объединяет публичный сайт рекламного агентства, внутреннюю CRM, расчёты, коммерческие предложения, заказы, производство, монтаж и финансы.

## Репозитории

### Основной

`deputat36/lider-bsk`

Содержит:

- публичный сайт;
- CRM;
- документацию;
- миграции и описание Supabase;
- интеграции сайта и CRM.

### Дополнительный

`deputat36/lidercalculator`

Используется только для самостоятельного калькулятора. Исторически в нём находится CRM v2 и CRM v4. CRM v4 переносится в основной репозиторий без немедленного удаления старой версии.

## Публичный сайт

Публичные страницы находятся в корне репозитория. Формы отправляют заявки в Supabase через публичную серверную функцию. Клиентский код не должен иметь административных ключей.

## CRM

Новая CRM размещается в каталоге `crm/`.

Планируемый основной адрес после проверки:

`/crm/`

На этапе переноса старая CRM в `lidercalculator` остаётся доступной как резервная версия.

## Архитектурные требования

- чистый HTML, CSS и JavaScript без обязательного сборщика;
- модульная структура JavaScript;
- Supabase Auth и RLS;
- мобильная адаптация;
- постепенное развитие через отдельные pull request;
- сохранение работающего функционала во время переноса.
