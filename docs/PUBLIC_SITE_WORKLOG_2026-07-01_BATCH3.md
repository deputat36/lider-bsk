Дата: 2026-07-01.
Ветка: `public-banner-seo-20260701`.
Контур: только публичный сайт РА «Лидер».

## Что сделано

Добавлены две новые SEO-посадочные страницы из баннерного кластера:

1. `pechat-bannerov-borisoglebsk.html`
   - печать баннеров;
   - фасадные баннеры;
   - баннеры для акции, открытия и мероприятия;
   - расчёт по размеру, макету, люверсам и срокам.

2. `banner-dlya-magazina-borisoglebsk.html`
   - баннер для магазина;
   - баннер для акции, открытия, распродажи;
   - временная реклама на фасаде или у входа;
   - связка с оформлением входа и витрины.

## Что есть на страницах

- SEO title и description.
- Canonical URL.
- Open Graph и Twitter Card.
- JSON-LD `Service` + `LocalBusiness`.
- Коммерческий первый экран.
- CTA на расчёт.
- Блоки: какие задачи закрывает баннер, что влияет на стоимость, ориентиры по цене, что прислать для расчёта, FAQ.
- Внутренние ссылки на смежные услуги.
- Общий виджет заявки `assets/public-lead-form.js?v=4`.
- Локальный prefill формы под услугу `Баннер` без изменения Supabase Edge Function.

1. Открыть обе страницы на desktop и mobile.
2. Проверить, что форма отображается корректно.
