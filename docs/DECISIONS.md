# Решения проекта

## 1. Основной репозиторий

Основной репозиторий проекта — `deputat36/lider-bsk`.

В нём размещаются публичный сайт, CRM, документация и интеграции.

## 2. Каталог CRM

CRM v4 размещается в каталоге `crm/`.

Публичные страницы остаются в корне сайта. Ресурсы CRM хранятся внутри `crm/assets/`.

## 3. Supabase

Данные РА «Лидер» используют объекты с префиксом `leader_`.

Объекты с префиксом `nav_` относятся к отдельному проекту.

## 4. Перенос CRM

Перенос выполняется поэтапно:

1. Копия CRM v4 добавляется в основной репозиторий.
2. Проверяется новый адрес.
3. Действующая версия сохраняется до завершения проверки.
4. Последующая очистка выполняется отдельной задачей.

## 5. Технологии

CRM v4 использует HTML, CSS, модульный JavaScript и Supabase.
