# Статус проекта РА «Лидер»

Дата обновления: 14 июня 2026 года.

## Работает

### Публичный сайт

- страницы услуг;
- формы заявок;
- отправка заявок в Supabase;
- мобильная адаптация;
- аналитика источников и UTM.

### CRM

В репозитории `deputat36/lidercalculator` работают CRM v2 и CRM v4.

CRM v4 включает:

- авторизацию;
- список и карточку заявок;
- потребности клиента;
- сохранённые расчёты;
- коммерческие предложения;
- создание заказа из согласованного КП.

CRM v4 переносится в каталог `crm/` основного репозитория. Существующий адрес пока сохраняется как резервный.

### Supabase

Рабочий проект: `ofewxuqfjhamgerwzull`.

Данные РА «Лидер» используют таблицы `leader_*`.

14 июня 2026 года применены миграции:

- `leader_protect_profile_roles_and_permissions`;
- `leader_fix_profile_guard_actor_detection`;
- `leader_remove_permissive_needs_policy_and_anon_order_rpc`.

Результат:

- сотрудник не может назначить себе административную роль;
- сотрудник не может менять собственные права и активность;
- администратор не может изменять владельца;
- доступ к потребностям использует проверку прав;
- создание заказа доступно только после входа.
