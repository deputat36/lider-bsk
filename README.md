# РА «Лидер»

Основной репозиторий проекта РА «Лидер».

## Что хранится в репозитории

- публичный сайт РА «Лидер»;
- будущая основная CRM;
- документация проекта;
- правила разработки;
- миграции Supabase после переноса.

## Важные документы

- `AGENTS.md` — правила работы для разработчиков и ИИ-помощника;
- `docs/PROJECT.md` — описание проекта и структуры;
- `docs/STATUS.md` — текущий статус;
- `docs/DECISIONS.md` — архитектурные решения.

## Репозитории

Основной репозиторий:

`deputat36/lider-bsk`

Дополнительный репозиторий калькулятора:

`deputat36/lidercalculator`

CRM v4 временно находится в `lidercalculator` и должна быть перенесена в этот репозиторий в каталог `crm/`.

## Правила базы данных

- таблицы CRM «Лидер» используют префикс `leader_`;
- таблицы `nav_*` относятся к проекту «Навигатор сделок»;
- данные разных проектов не смешиваются;
- изменения структуры базы выполняются через миграции;
- секретные ключи не хранятся в GitHub.
