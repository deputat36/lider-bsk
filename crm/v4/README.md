# CRM v4 РА «Лидер»

Точка входа новой CRM в основном репозитории:

`crm/v4/index.html`

## Текущий этап переноса

Интерфейс, мобильный навигатор и точка входа находятся в `deputat36/lider-bsk`.

Проверенные рабочие модули CRM временно загружаются из:

`deputat36/lidercalculator/assets/v4/`

Это переходное решение позволяет:

- не останавливать действующую CRM;
- проверить новый адрес и мобильный интерфейс;
- сохранить старый адрес как резерв;
- перенести модули физически отдельным безопасным этапом.

## Что нельзя делать до завершения проверки

- не удалять CRM v4 из `lidercalculator`;
- не менять Supabase-проект;
- не удалять таблицы и функции `leader_*`;
- не использовать таблицы `nav_*`.

## Рабочий сценарий

`заявка → потребность → расчёт → КП → заказ`
