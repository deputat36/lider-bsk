Политика обработки персональных данных

Настоящая политика описывает порядок обработки данных, которые пользователь передает через сайт РА «Лидер».

Какие данные могут обрабатываться

Через форму заявки могут передаваться имя, телефон, выбранная услуга, комментарий к заказу, параметры заказа и адрес страницы, с которой отправлена заявка.

Цель обработки

Данные используются для связи с клиентом, уточнения задачи, расчета стоимости, подготовки предложения и дальнейшего сопровождения заявки.

Передача данных

Данные не продаются третьим лицам. При необходимости отдельные сведения о заказе могут передаваться подрядчикам только для расчета или выполнения работ, без лишней персональной информации.

Хранение данных

Заявки могут храниться в CRM рекламного агентства для обработки обращений, истории заказов и повторной связи с клиентом.
